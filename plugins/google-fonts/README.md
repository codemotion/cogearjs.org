# Cogear.JS plugin **google-fonts**

About this plugin.

# Installation

Go to the site folder and install plugin with command:
```bash
yarn add cogear-plugin-google-fonts
# or 
npm install cogear-plugin-google-fonts
```

Plugin will loads up automatically.
